🛠️ Product Manager

Yahan se naye products add/edit karein. Ye sab aap ke browser mein chalta hai — koi internet ya server zaroori nahi.

  1. Form bharein — naam, category, price aur photo choose karein, phir “Product Add” dabayein.
  2. Jitne products chahein add karte jayein (niche list mein nazar aayenge).
  3. Jab tayyar ho jayein, “⬇ Download products.js” dabayein.
  4. Download hui file ko js folder mein purani products.js ki jagah rakh dein.
  5. Website refresh karein — naye products live! 🎉

💡 Logo, brand naam, WhatsApp number, Instagram ya colors badalne ke liye niche Website Settings use karein.

⚙️ Website Settings — Logo, Brand & Contact

Yahan se logo, naam, WhatsApp number, Instagram aur colors change karein. Baad mein niche “⬇ Download config.js” dabayein aur us file ko js folder mein purani config.js ki jagah rakh dein.
